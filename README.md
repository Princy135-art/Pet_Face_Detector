# 🐾 Pet Landmark Detection using AI

## 📌 Project Description
This project is an AI-based application that detects landmarks (key points) on images using a deep learning model. It uses a pose estimation model to identify keypoints such as eyes, nose, and other important regions.

## 🚀 Features
- Detects keypoints (landmarks) on images
- Uses pre-trained AI model
- Simple and easy to run
- Visualizes landmarks using OpenCV

## 🧠 Technologies Used
- Python
- OpenCV
- Ultralytics YOLOv8
- Jupyter Notebook
